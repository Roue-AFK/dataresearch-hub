# UI大改版方案 v15.0：多平台数据调研集合站

## 用户目标
做一个**抖音 + 闲鱼 + 小红书**三平台数据调研集合站，用 AI 辅助调研、总结、拟稿。

## 各平台核心功能

| 平台 | 核心功能 |
|------|---------|
| 🐟 闲鱼 | 商品采集、价格分析、文案分析、AI调研（已有） |
| 🎵 抖音 | 热门话题追踪、AI调研总结、拟稿话术 |
| 📕 小红书 | 笔记/爆文分析、关键词趋势追踪、AI调研 |

## 配色方案：Slate Professional + 暖橙

暗色主题（默认）：
- 背景: #0F172A（午夜板岩）
- 侧边栏: #1E293B
- 卡片/面板: #1E293B
- 边框: #334155
- 强调色: #F59E0B（暖橙，闲鱼品牌关联）
- 辅助色: #38BDF8（天空蓝，数据）、#10B981（翡翠绿，成功）
- 文字: #F1F5F9 / #94A3B8

3. 抖音话题追踪模块（新建）
4. 小红书笔记分析模块（新建）
